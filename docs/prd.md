## 4. 业务规则与逻辑

- 数字分身聊天区无需登录，任何访客均可直接提问。
- 聊天记录仅在当前会话内保留，刷新页面后重置，无需持久化存储。
- 数字分身回复优先匹配预设问答（关键词模糊匹配），无匹配时基于知识库自由生成回复。
- 页面为单页滚动结构，导航栏各项点击后平滑滚动至对应区块。
- 手机端导航栏折叠为汉堡菜单。
